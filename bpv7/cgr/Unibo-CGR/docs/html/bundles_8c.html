initialize_bundle
Date Written:
31/03/20
Returns
int
Return values
0Success case: all fields OK
-1Bundle: NULL
-2Bundle's fields are not valid
-3Bundle's failedNeighbors/geoRoute: NULL
Parameters
[in,out]*bundleThe bundle for which we want to set all the fields
[in]backward_propagationboolean: 0 if the sender_node has to be into the excludedNeighbors list, 1 otherwise.
[in]criticalboolean: 1 the bundle is critical -> it will be send to all "viable" neighbors, 0 otherwise -> it will be send to only one neighbor (best route).
[in]dlvConfidenceThe previous delivery confidence of the bundle (from 0.0 to 1.0)
[in]expiration_timeThe deadline
[in]priorityThe priority of the bundle: Bulk, Normal or Expedited
[in]ordinalOnly for expedited priority, from 0 to 255
[in]probeboolean: 0 if it's a probe bundle, 1 otherwise
[in]fragmentableboolean: 1 if this bundle is fragmentable, 0 otherwise
[in]sizeThe sum of payload + header
[in]sender_nodeThe node that directly sent to us this bundle
[in]terminus_nodeThe node to which this bundle has to be forwarded

int set_failed_neighbors_list ( CgrBundle *  bundle,
unsigned long long  ownNode 
)

Build the bundle's failedNeighbors list from the bundle's geoRoute list.

Function Name:
set_failed_neighbors_list
Date Written:
29/03/20
Returns
int
Return values
>= 0The total number of failed neighbor found, the real number of failed neighbors could be less than this return value due to some nodes duplicate or more.
-1Arguments error
-2MWITHDRAW error
Parameters
[in]*bundleThe bundle to which we want to set the failed neighbors list. The caller must initialize the geographic route of the bundle (geoRoute).
[in]ownNodeThe own node (ipn node)
Notes:
  1. A failed neighbor is a node that appears in the geographic route of the bundle immediately after the ownNode. So there was a loop and the ownNode previously send this bundle to that neighbor.
  2. If this function returns a value greater than 0 you can assume that there was a loop
  3. The bundle->failedNeighbors list initially will be resetted.
