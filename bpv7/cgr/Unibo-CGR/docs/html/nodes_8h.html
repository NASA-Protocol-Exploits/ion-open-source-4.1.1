insert_neighbors_to_reach_destination
Date Written:
28/04/20
Returns
int
Return values
>= 0The neighbors inserted into the list
-2MWITHDRAW error
-3Arguments error
-5Local node's neighbors list not initialized
Parameters
[in]neighborsThe neighbors list to build the destination's neighbors list. Each data in this list is an unsigned long long (the ipn number of the neighbor)
