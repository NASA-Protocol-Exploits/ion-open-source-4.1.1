Delete the route passed as argument and manage all its references with other routes, with all the contacts in the hops list and with the list where it was putted in (referenceElt).

Function Name:
delete_cgr_route
Date Written:
21/01/20
Returns
void
Parameters
[in]*dataThe Route that we want to delete
Notes:
  1. All contacts in the hops list at the end will not have anymore any reference with this route.
  2. The route will be removed also from the list where it was putted in (referenceElt field)
  3. For the other references with other routes see the update_references function
