Moderate Source Routing: this function checks if the route stored in bundle->msrRoute is viable for the forwarding of the bundle. In affermative case then we propose this route as "best route".

Function Name:
tryMSR
Date Written:
23/04/20
Returns
int
Return values
>= 0The number of "best routes" found.
-1The MSR route stored in the bundle isn't viable.
-2MWITHDRAW error
-3Arguments error
-4Critical bundle, MSR is not optimal for a critical bundle so we don't perform the MSR algorithm
Parameters
[in]current_timeThe current (differential from interface) time from the start of Unibo-CGR
[in]*bundleThe bundle to forward
[in]excludedNeighborsThe excluded nodes list SABR 3.2.5.2
[in]file_callThe file where we print some informations about what has been done during the MSR algorithm.
[out]*bestRoutesThe MSR's output, the list of "best routes" if there are any.
