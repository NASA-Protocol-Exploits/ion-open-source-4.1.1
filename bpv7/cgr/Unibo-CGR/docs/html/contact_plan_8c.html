Add a contact to the contacts graph.

Function Name:
addContact
Date Written:
23/01/20
Returns
int
Return values
2Success case: Revised contact's xmit rate (and MTVs)
1Success case: The contact now is inside the contacts graph
0Arguments error case: The contact cannot be inserted with this arguments
-1The contact cannot be inserted because it overlaps some other contacts.
-2MWITHDRAW error
-3Precondition error, you must call initialize_contact_plan() before!
Parameters
[in]fromNodeThe contact's sender node
[in]toNodeThe contact's receiver node
[in]fromTimeThe contact's start time
[in]toTimeThe contact's end time
[in]xmitRateIn bytes per second
[in]confidenceThe confidence that the contact will effectively materialize
[in]copyMTVSet to 1 if you want to copy the MTV from mtv input parameter. Set to 0 otherwise
[in]mtvThe contact's MTV: this must be an array of 3 elements.
