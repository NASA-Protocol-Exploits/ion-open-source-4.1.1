◆ REVISABLE_RANGE

#define REVISABLE_RANGE   1

Set to 1 if you want to permits that range's owlt can be changed. Set to 0 otherwise.

You should consider the owlt as contact plan changes.

Notes:
  1. I suggest you to set this macro to 1 if you copy ranges from another contact plan by an interface.
