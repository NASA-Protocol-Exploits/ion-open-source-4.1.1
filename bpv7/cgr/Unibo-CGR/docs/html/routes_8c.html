static void update_references ( Route *  route)
static

Manage all the references between this route and other related routes (father, children, selectedFather, selectedChild)

Function Name:
update_references
Date Written:
21/01/20
Returns
void
Parameters
[in]*routeThe Route that we want to be "forgotten" by all its children, father selectedFather and selectedChild
Notes:
  1. The children of this route at the end will have the father field setted to NULL.
  2. The father of this route at the end will not see anymore this route in the children list.
  3. The selectedFather of this route at the end will acquire as selectedChild the selectedChild of this route.
  1. The selectedChild of this route at the end will acquire as selectedFather the selectedFather of this route.
  2. If this route has a selectedChild field setted to NULL we set to 0 the spursComputed field of the selectedFather.
